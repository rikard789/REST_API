Welcome to my website

1. On this website you can add and remove words from collection.
You can add the word by adding in browser search /add/wordYouWantToAdd
For example: localhost:3000/add/car

You can remove the word by adding in browser search /remove/wordYouWantToRemove
For example: localhost:3000/remove/car


2. You can also count appearances of the same word.

You count appearances of the same word by adding in browser search /count/wordYouWantToCount
For example: localhost:3000/count/car


3. You can also see all unique words in the collection or all the words in the collection

You see all unique words by adding in browser search /showUniq
For example: localhost:3000/showUniq

You see all words by adding in browser search /show
For example: localhost:3000/show
